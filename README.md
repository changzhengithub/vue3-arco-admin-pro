# arco-admin-pro

基于Vue3+Vite+Ts+Arco 构建的后台管理系统。

## IDE设置

新版本只需安装 Vue - Official 插件即可，Volar Takeover模式也无需开启，把TypeScript and JavaScript Language Features服务启动。

## 代码检查

格式化代码安装prettier插件，如需自动代码检查请安装Eslint插件。

## 项目构建

```sh
npm install
npm run dev
npm run build-only
```
